Feature: Create a new kanban board

Create the first board to the cleanKanban system

Given a registered user Teddy has logged in into the cleanKanban system
When Teddy creates a board named "Scrum Board"
Then the "Scrum Board" should be added to Teddy's personal board
And the "Scrum Board" should contains four Stages: Backlog, Archive, IceBox, TrashCan
